# PPW Portfolio Website

A simple multi-page personal portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Home page
- `about.html` - About section
- `education.html` - Education section
- `projects.html` - Projects section
- `skills.html` - Skills section
- `certificates.html` - Certificates section
- `contact.html` - Contact page
- `style.css` - Site styling
- `script.js` - JavaScript interactions
- `images/` - Image assets
- `ChandanaKRresume.pdf` - Resume file

## Usage

1. Open `index.html` in a web browser.
2. Navigate through the site using the available links.

## Notes

- This is a static website and does not require a server to run.
- Customize the content, styles, or scripts as needed.
